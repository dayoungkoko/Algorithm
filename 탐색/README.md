# 1. 순차 탐색
- 데이터 정렬 여부와 무관하게 앞에서부터 데이터를 하나씩 차례대로 확인하는 방법
- 최악의 경우 시간 복잡도는 $O(N)$

# 2. 이진 탐색
- 데이터가 정렬되어 있을 때만 가능하며 대용량 데이터 처리에 유용하다.
- 시작점, 끝점, 중간점을 설정하여 탐색 범위를 절반씩 좁혀가며 데이터를 탐색한다
- 시간 복잡도는 $O(logN)$
